Official Websites#

IRCC Related Services#

Immigration, Refugees and Citizenship Canada (IRCC) is responsible for immigration, refugees, and citizenship services in Canada.

Official Website of IRCC

Access official information of Immigration, Refugees and Citizenship Canada (IRCC).

https://www.canada.ca/en/immigration-refugees-citizenship.html

GCKey

Access secure online services with GCKey.

https://www.canada.ca/en/immigration-refugees-citizenship/services/application/account.html

Visa Status Tracker

Check the status of your immigration or citizenship application.

https://www.canada.ca/en/immigration-refugees-citizenship/services/application/check-status.html

Processing Times

View the processing times for various applications.

https://www.canada.ca/en/immigration-refugees-citizenship/services/application/check-processing-times.html

IRCC Forms

Find forms and guides for various applications.

https://www.canada.ca/en/immigration-refugees-citizenship/services/application/application-forms-guides.html

Official Website of CBSA

Find information on border services of Canada Border Services Agency (CBSA).

https://www.cbsa-asfc.gc.ca/menu-eng.html

ATIP Related Services#

Access to Information and Privacy (ATIP) requests allow individuals to access personal information held by government institutions.

IRCC ATIP Request

Request your personal information from IRCC.

https://www.canada.ca/en/immigration-refugees-citizenship/corporate/transparency/access-information-privacy.html

CBSA ATIP Request

Access your personal information held by the Canada Border Services Agency.

https://www.canada.ca/en/immigration-refugees-citizenship/corporate/transparency/access-information-privacy/request-privacy-act.html

CSIS ATIP Request

Request personal information from the Canadian Security Intelligence Service.

https://www.canada.ca/en/immigration-refugees-citizenship/corporate/transparency/access-information-privacy/request-privacy-act.html

Express Entry (EE)#

Express Entry is a fast-track immigration system for skilled workers, professionals, and international students.

Federal Skilled Worker Program

Apply through the Federal Skilled Worker Program for fast-tracked immigration.

https://www.canada.ca/en/immigration-refugees-citizenship/services/immigrate-canada/express-entry/eligibility/federal-skilled-workers.html

Canadian Experience Class

For those with Canadian work experience seeking permanent residence.

https://www.canada.ca/en/immigration-refugees-citizenship/services/immigrate-canada/express-entry/eligibility/canadian-experience-class.html

Federal Skilled Trades Program

For skilled workers in specific trades seeking permanent residence.

https://www.canada.ca/en/immigration-refugees-citizenship/services/immigrate-canada/express-entry/eligibility/skilled-trades.html

The Provincial Nominee Program (PNP) allows Canadian provinces and territories to nominate individuals for immigration based on their specific economic needs.

British Columbia (BC PNP)

Learn about the British Columbia Provincial Nominee Program.

https://www.welcomebc.ca/immigrate-to-b-c/about-the-bc-provincial-nominee-program?

Ontario Immigrant Nominee Program (OINP)

https://www.ontario.ca/page/ontario-immigrant-nominee-program

Alberta Immigrant Nominee Program (AINP)

Find out about the Alberta Immigrant Nominee Program.

https://www.alberta.ca/alberta-advantage-immigration-program

Manitoba Provincial Nominee Program (MPNP)

Discover the Manitoba Provincial Nominee Program.

https://immigratemanitoba.com/immigrate/

Nova Scotia Nominee Program (NSNP)

Check the Nova Scotia Nominee Program.

https://liveinnovascotia.com/

Saskatchewan Immigrant Nominee Program (SINP)

Learn about the Saskatchewan Immigrant Nominee Program.

https://www.saskatchewan.ca/residents/moving-to-saskatchewan/live-in-saskatchewan/by-immigrating/saskatchewan-immigrant-nominee-program?

Newfoundland and Labrador Provincial Nominee Program (NLNP)

Explore the Newfoundland and Labrador Provincial Nominee Program.

https://www.gov.nl.ca/immigration/immigrating-to-newfoundland-and-labrador/provincial-nominee-program/overview/?

Prince Edward Island Provincial Nominee Program (PEI PNP)

Learn about the Prince Edward Island Provincial Nominee Program.

https://www.princeedwardisland.ca/en/topic/immigrate-to-pei-as-a-worker

New Brunswick Provincial Nominee Program (NB PNP)
