==================================================================
How to Apply for a Canadian Visitor Visa - Application Process
==================================================================

This guide outlines the step-by-step process for applying for a Canadian visitor visa.

Before You Apply
----------------

Preparation Steps
~~~~~~~~~~~~~~~~~

1. Check if you need a visa
2. Gather required documents
3. Create digital copies of documents
4. Have valid credit card for payment
5. Access to scanner/camera for documents

Online Application Process
--------------------------

Creating Account
~~~~~~~~~~~~~~~~

1. Create GCKey or Sign-in Partner login
2. Complete personal profile
3. Start visitor visa application
4. Save reference number

Completing Application
~~~~~~~~~~~~~~~~~~~~~~

1. Fill all mandatory fields
2. Provide family information
3. Detail travel history
4. Enter employment/education details
5. Explain purpose of travel

Document Upload Requirements
----------------------------

Required Format
~~~~~~~~~~~~~~~

* PDF, JPEG, PNG, DOC, or DOCX
* Maximum file size: 4MB per file
* Clear, legible scans
* Color copies preferred

Document Checklist
~~~~~~~~~~~~~~~~~~

* Passport/travel document
* Photos meeting specifications
* Financial documents
* Purpose of travel proof
* Supporting documents
* Family information form

Biometrics Process
------------------

Appointment Booking
~~~~~~~~~~~~~~~~~~~

* Pay biometrics fee
* Receive instruction letter
* Book at authorized center
* Attend appointment

Collection Process
~~~~~~~~~~~~~~~~~~

* Fingerprints taken
* Photo taken
* Documents verified
* Receipt provided

After Submission
----------------

Application Timeline
~~~~~~~~~~~~~~~~~~~~

1. Application received
2. Initial review
3. Biometrics verification
4. Detailed review
5. Decision made
6. Passport request (if approved)

Status Updates
~~~~~~~~~~~~~~

* Check online account
* Processing times vary
* Additional documents may be requested
* Email notifications

If Approved
-----------

Next Steps
~~~~~~~~~~

1. Submit passport (if requested)
2. Visa stamped in passport
3. Review visa details
4. Plan travel within validity

If Refused
----------

Options Available
~~~~~~~~~~~~~~~~~

* Review refusal letter
* Address concerns
* Reapply if circumstances change
* Consider other options
